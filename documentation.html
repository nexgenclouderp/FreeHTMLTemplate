1. Overview:

Liner Pro is HTML template for professional business websites. It is retina ready, fully responsive, crystal clear line icons, developed on most popular - Bootstrap Framework which fulfills all your website needs.

Key Features:

  • 3 hero section layouts: Image, Slider and Video
  • 4 color options provided
  • Custom color option using CSS Variables, just provide your color once and whole styles will be of your color!
  • Based on Bootstrap Latest Version v3.3.7
  • HTML5, CSS3, jQuery
  • 100% Responsive
  • Working Contact Form With Ajax with Simple Email Template
  • Easy to Customize
  • Valid HTML5
  • Minimal and Clean
  • HTML, CSS, JS Included
  • Fonts Awesome Icons
  • Linea Icons
  • Free Google Fonts
  • All files are well commented
  • Displays well in all modern browsers and devices

Custom Theme CSS:

Liner Pro uses CSS variables for custom colored theme. You just need to specify your color for variable inside assets/css/custom-theme.css file and the whole theme changes to your color.

:root{ /* Any HEX or RGB color code */ --custom-color: #daa520; }

Email Messages:

Liner Pro uses a very elegant Email template. You need to specify your email address where all the Emails will be sent when filling out the contact form. Specify your email address in mail.php file.

$to = 'youremail@domain.com'; // put your email here
